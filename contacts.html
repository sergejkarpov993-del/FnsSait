FAQ

Часто задаваемые
вопросы

Как быстро можно начать работу? +
После подписания договора и оплаты аккаунт активируется в течение одного рабочего дня. Базовую настройку профиля, источников и уведомлений наш менеджер проводит в режиме онбординг-звонка — занимает 45–60 минут.
Что входит в 30-дневный пилот? +
Полный функционал выбранного тарифа, неограниченное количество тестовых пользователей, персональная настройка источников под вашу отрасль, еженедельные check-in звонки с нашей командой и вся поддержка без ограничений. Без предоплаты.
Можно ли подключить нестандартный источник? +
Да. Мы добавляем новые источники по запросу клиента. Срок — 5 рабочих дней для большинства официальных сайтов регуляторов. Для нестандартных форматов (PDF с закрытых порталов, специализированные базы данных) обсуждаем индивидуально.
Какой формат оплаты? +
Годовой контракт с оплатой по счёту в рублях. Предусмотрена рассрочка поквартально. При оплате за весь год вперёд — скидка 10%. Работаем с российскими юрлицами, оформляем все закрывающие документы.
Как устроены уведомления для команды? +
Каждый пользователь настраивает собственный профиль: выбирает регуляторы, теги, приоритеты и канал доставки (email, Telegram, Teams, Slack). Администратор может управлять настройками всей команды централизованно и задавать роли с разными уровнями доступа.
Есть ли интеграция с нашей GRC-системой? +
На тарифе Enterprise доступен полноценный REST API с подробной документацией. Поддерживается экспорт событий в JSON, возможна настройка вебхуков для передачи данных в стороннюю систему в режиме реального времени. Готовые коннекторы — по запросу.
